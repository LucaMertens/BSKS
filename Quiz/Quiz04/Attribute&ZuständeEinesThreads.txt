In einem Betriebssystem, das kein Multi-Threading unterstützt, werden unter anderem die folgenden Attribute für jeden Prozess in der Prozesstabelle gespeichert. Welche der genannten Attribute müssen in einem Betriebssystem, das Multi-Threading unterstützt, für jeden Thread eines Prozesses separat gespeichert werden?


[ ] Parent Process ID (ID des Prozesses, der den aktuellen Prozess abgespaltet hat)

[ ] User ID (ID des Benutzers, der den Prozess gestartet hat)
	
[ ] Code-Speicherbereich (Zeiger auf den Beginn des Codes im Speicher)

[X] Stack Pointer (Zeiger auf das aktuelle Stack Frame auf dem Laufzeit-Stack)
	
[X] Program Counter (Adresse des als nächstes auszuführenden Befehls)
